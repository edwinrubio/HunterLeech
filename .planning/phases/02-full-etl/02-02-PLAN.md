SIGEP pitfall — entidad_codigo constraint:
  SIGEP has no codigo_entidad field. MERGE (ent:EntidadPublica {codigo_entidad: null}) would
  violate the constraint. SIGEP entities MUST MERGE on nombre, not codigo_entidad.
  Use: MERGE (ent:EntidadPublica {nombre: row.nombre_entidad})

SIRI pitfall — trailing whitespace:
  numero_identificacion arrives as "7534386        " (padded).
  ALWAYS: cedula_raw = (row.get("numero_identificacion") or "").strip()

SIGEP pitfall — nombre field:
  SIGEP nombre field = numerodeidentificacion (privacy redaction). NEVER write to Persona.nombre.

SIRI — DD/MM/YYYY date format:
  fecha_efectos_juridicos is "22/04/2005". Store as string. Use numero_siri for pagination order.

Multas — no document type discriminator:
  classify_contratista_type() uses name-based heuristics (legal suffixes).

```python
def _parse_salario(raw) -> float | None:
    """SIGEP uses commas as thousands separator: '1,440,300' -> 1440300.0"""
    if raw is None:
        return None
    try:
        cleaned = str(raw).replace(",", "").strip()
        return float(cleaned)
    except (ValueError, TypeError):
        return None
```

```python
_EMPRESA_SUFFIXES = {
    "s.a.s", "sas", "s.a", "ltda", "s.c.a", "e.u",

def classify_contratista_type(nombre: str | None) -> str:
    """Classify Multas contractor as empresa or persona based on name heuristics.
    No document-type discriminator exists in this dataset (pitfall 8).

    if not nombre:

    key = nombre.lower().strip()
    for suffix in _EMPRESA_SUFFIXES:
        if suffix in key:

    return "persona"
```
