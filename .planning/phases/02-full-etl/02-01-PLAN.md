**_parse_valor_secop2() helper:**
```python
def _parse_valor_secop2(raw) -> float | None:
    """SECOP II values are plain integer strings — no dot removal."""
    if raw is None:
        return None
    try:
        return float(str(raw).strip())
    except (ValueError, TypeError):
        return None
```

Do NOT reuse `_parse_valor` from secop_integrado.py — it strips dots (SECOP I format). SECOP II uses plain integers.
